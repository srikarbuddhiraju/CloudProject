# Contributing

This project is in early planning/pre-build. **External pull requests are
not currently accepted** — there is no Developer Certificate of Origin (DCO)
or contributor agreement process in place yet, which is required before
accepting outside contributions under the project's
[fair-source license](LICENSE).

Issues and discussion are welcome.

A DCO/CLA process will be set up before the first external PR is reviewed.
This file will be updated when that happens.
